## 学び
- bundler  
  参考サイト  
  [Ruby,Rails,gem,bundlerの基本関係](https://www.sejuku.net/blog/15339)  
  [bundlerについて基本解説](https://pikawaka.com/rails/bundler)

  - gem…gemには2つの意味があるため分けて記述する。
    - gem①…他の開発者が作製したパッケージ。  
  例えばユーザー登録機能や認証機能などはdeviceというgemが公開されているためこれを利用することで簡単に自分の環境でも実装が可能となる。  
  参考サイト…[deviceについて](https://www.sejuku.net/blog/13378)

    - gem②…gemを管理するためのパッケージ管理ツールのこと。(gem(バージョンとかの情報)をgem(パッケージ管理ツール)で管理するっていうきもい関係)今はbundlerに代わってきている。
  
  - bundler…gem管理ツールの1つで現在最も主流。gemの依存関係を記述できるため、複数のgemを依存関係を考慮して使うことができるのが利点である。以下のコードをターミナルで実行してダウンロードが可能。
  ```
  gem install bundler  #最新のbundlerをダウンロード
  bundler -v  #bundlerのバージョンを確認

  ```
  Ruby7.0以降はscssファイル追加後にサーバーを起動しようとするとエラーが生じる。それは今までは自動で行われていたgemのインストールが行われなくなったためである。そのためGemfileを後述のサイトにしたがって修正し、"bundle install"を実行する必要が生じた。そのため今回bundlerについて調べることになったのである。  
  参考サイト…[cannot load such file --sasscのエラー解決法](https://qiita.com/Yu_unI1/items/15926b72934dada66ed3)
  
  - Ruby on Rails…WEBアプリを開発する際に使用するフレームワーク（枠組み）のこと。これはフレームワークであり記述する際に使う言語はRubyである。
  - uru…Rubyのバージョンを切り替えるためのツール。  
    参考サイト…[Rubyバージョン切り替え](https://www.sejuku.net/blog/12327)
